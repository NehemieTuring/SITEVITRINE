# Winconcept Industry - Creative Studio Frontend

This is a premium, multiservice creative studio frontend built with React 19, TypeScript, and Tailwind CSS.

## 🚀 Fast Start

1.  **Extract/Move files** to your local environment.
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run development server**:
    ```bash
    npm run dev
    ```
4.  **Build for production**:
    ```bash
    npm run build
    ```

## 🛠 Tech Stack

- **React 19**: Latest React features for maximum performance.
- **Vite**: Rapid build tool for optimized development.
- **Tailwind CSS**: Utility-first styling with a custom brand palette.
- **Framer Motion**: State-of-the-art animations for immersive experience.
- **React Router DOM**: Seamless client-side navigation.
- **Lucide React**: Modern, consistent iconography.

## ✨ Key Features

- **Full Internationalization (i18n)**: 
  - Dynamic language switching (FR/EN).
  - Browser language detection.
  - Multi-module translation system based on Context API.
- **Premium Dark/Light Mode**: 
  - Custom brand colors (Deep Blue & Gold).
  - Smooth transitions and system preference detection.
- **Custom Universal Player**: 
  - Advanced audio/video player with custom controls.
  - Auto-hide UI after 4 seconds of inactivity.
  - Fullscreen and volume management.
- **Responsive & Immersive UX**: 
  - Framer Motion page transitions.
  - Skeleton loaders for asynchronous data simulation.
  - Dynamic service filtering and lightbox gallery.
  - Mobile-first, fully responsive design system.

## 📂 Project Structure

- `src/components/layout`: Main shell, Navigation, Footer, and transitions.
- `src/components/media`: Custom audio/video players and galleries.
- `src/components/ui`: Custom ShadCN-inspired components and skeletons.
- `src/contexts`: Global state for Theme, Language, and Media.
- `src/locales`: Multi-language JSON modules.
- `src/pages`: Fully implemented pages (Home, Services, Portfolio, etc.).
- `src/utils`: API mocking and helper functions.

## 🎨 Design System

- **Primary**: Deep Blue `#0A2F44` (Stability & Professionalism).
- **Secondary (Accent)**: Gold `#C8A252` (Luxury & Creativity).
- **Typography**: 
  - Titles: *Playfair Display* (Elegant Serif).
  - Body: *Poppins* (Modern Sans-serif).
- **Spacing**: 4px / 8px grid based system.
